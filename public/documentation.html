This API keeps a database of movies for people to search through including information such as the directors, cast, and crew.

Request description The endpoint URL Method Request data Response data
Get a list of all the movies /movies GET n/a A JSON array of objects containing data of all the movies of the database
Get data about movie by title /movies/[title] GET n/a A JSON object containing data of the specified movie
Example:
{
"id": "6",
"title": "Psycho",
"director": "Alfred Hitchcock",
"year": "1960"
}
