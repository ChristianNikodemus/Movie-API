This API keeps a database of movies for people to search through including information such as the directors, cast, and crew.

Request description The endpoint URL Method Request data Response data
Welcome user with message / GET n/a A string containing the greeting message Example:
Welcome to my movie API Database!

Example:
{

"Title": "Pulp Fiction",
"Description": "The lives of two mob hitmen, a boxer, a gangster and his wife, and a pair of diner bandits intertwine in four tales of violence and redemption.",
"Year": "1994",

"614018348cb22d9e16165cb3",
"614018468cb22d9e16165cb4"
],
"Director": [
"6140196f8cb22d9e16165cbf"
],
"ImagePath": "pulpfiction.png",
"Featured": true
}

Return data about a genre (description) by name/title (e.g., “Thriller”) /genres/[title] GET n/a A JSON object containing data of the specified genre
Example:
{
"_id": "614018348cb22d9e16165cb3",
"Title": "Crime",
"Description": "Crime fiction, detective story, murder mystery, mystery novel, and police novel are terms used to describe narratives that centre on criminal acts and especially on the investigation, either by an amateur or a professional detective, of a serious crime, generally a murder."
}
Return data about a director (bio, birth year, death year) by name /directors/[name] GET n/a A JSON object containing data of the specified genre
Example:
{
"title": Crime,
"description": "Crime fiction, detective story, murder mystery, mystery novel, and police novel are terms used to describe narratives that centre on criminal acts and especially on the investigation, either by an amateur or a professional detective, of a serious crime, generally a murder."

Allow new users to register /register POST A JSON object containing data of the new user
Example:
{
"name": "Nick Crawford",
"username": "Nickypoo62",
"email": "nick.crawford@gmail.com",
"password": "catsaretheworst"
} 		A JSON object containing data of the new user that was added
Example:
{
"name": "Nick Crawford",
"username": "Nickypoo62",
"email": "nick.crawford@gmail.com",
"password": "catsaretheworst"
}
Allow users to update their user info (username) /register/[username] PUT A JSON object containing data of the updated username
Example:
{
"username": "Chrissy",
} 		A return message communicating the success or failure of the change to username
Example:
Your username has now been changed to: Chrissy
Allow users to add a movie to their list of favorites /favourites POST A JSON object containing data of the new favourite movie entry
Example:
{
"title": "Pirates of the Caribbean",
} 		A return message communicating that the new favourite movie entry has been added
Example:
Pirates of the Caribbean Has been added to your favourites!
Allow users to remove a movie from their list of favorites /favourites/[title] DELETE A JSON object containing the data of the movie that will be deleted A return message communicating that the movie has been deleted
Example:
The movie: Pulp Fiction was deleted.
Allow existing users to deregister /userAccounts/[name] DELETE A JSON object containing a username in which will delete the associated account A return message communicating that the user has been deleted
Example:
The account belonging to: Christian Nikodemus was deleted.
